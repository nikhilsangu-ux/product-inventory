# Product Inventory System

Full Stack Inventory Management System using Node.js, Express, MongoDB and HTML.

## Features

Add Product  
View Products  
Update Quantity  
Low Stock Detection  

Low stock rule:
quantity < 10

## Backend APIs

POST /products

GET /products

PUT /products/:id

GET /products?lowStock=true

## Technologies

Node.js  
Express  
MongoDB  
HTML + JavaScript

## Run Project

Start MongoDB

Run backend

node server.js

Open frontend/index.html
